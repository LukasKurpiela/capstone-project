# capstone-project CoinTrax
CoinTrax ist ein Kryptowährungs-Tracker mit dem sich das Marktgeschehen der Top 100 Kryptowährungen verfolgen lässt. Die eigenen Lieblingscoins können favorisiert und ein persönliches Portfolio inkl. Transaktionshistorie kann erstellt werden. Auf der Newsseite werden aktuelle Neuigkeiten angezeigt.
